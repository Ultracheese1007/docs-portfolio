# Xinmei Ma — Documentation Portfolio

A docs-as-code portfolio built with [Docusaurus](https://docusaurus.io). It contains three worked
samples of developer documentation:

- **Airflow PoC (case study)** — backend/data-platform architecture, canonical data models, and
  data-quality gates. Sanitised from a data-platform internship.
- **CityFlow API integration guide** — SDK-style onboarding for a Spring Boot + Kafka backend,
  including event-driven behaviour. Backed by a bundled OpenAPI spec.
- **Thesis pipeline reproducibility** — data flow, environments, and leakage-prevention rules.

## Run locally

```bash
npm install
npm start          # http://localhost:3000
```

## Build

```bash
npm run build      # outputs to ./build
npm run serve      # preview the production build
```
